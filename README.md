### Nextjs 연습 리파지토리

- nextjs에서 사용되는 구조와 생태계를 연습 합니다.
- SSR의 원리와 동작 방법을 구체적으로 접근 해 봅니다.
- 각종 라이브러리 연동 해 봅니다.
- pnp 적용을 통해 용량을 줄이도록 합니다.

---

### 1. 기본 라이브러리(계속 추가!)

```bash
1. 로그인 및 인증 : next-auth
2. 클라이언트 상태관리 : zustand
3. 스타일 : tailwind
4. 추가 예정...!
```

### 2. 앱 명령어

```bash
1. 개발 : yarn run dev
2. 빌드 : yarn run build
3. 실행 : yarn run start
```
